WCAG Color Comparison

Current vs proposed Machinon theme colors. Side-by-side on light and dark backgrounds.

Key change: Outline buttons use --main-text-color for text (dark on light, white on dark) instead of the semantic color. This guarantees AAA text contrast (17.4:1 / 7.5:1) regardless of theme. The semantic color is only used for the border (decorative) and filled button backgrounds.
