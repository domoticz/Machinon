Machinon Button Reference Design System v1.0

All button styles in the Machinon Domoticz theme. Hover over buttons to see their interactive states.

Convention: All Bootstrap semantic button classes (.btn-info, .btn-primary, .btn-success, .btn-warning, .btn-danger) are filled — colored background, white text. This matches Bootstrap's original design. Outline is reserved for toggle states (.btn-default, .btn-mini) and utility buttons (zoom, log-filter). Color differentiates meaning, not visual weight.

Filled Buttons

Colored background, white text. Used for standalone action buttons. Hover darkens via filter: brightness(0.85). All semantic color variants (success, warning, danger) are filled — color differentiates meaning, not visual weight.

Button Groups

Two styles based on context. Device cards use pills (wraps gracefully). Toolbars use connected groups (full-width, no wrapping).
